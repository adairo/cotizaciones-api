Cotizaciones

Up 
get /cotizaciones/{productSku}/{plazo}
(cotizacionesProductSkuPlazoGet)

Path parameters

productSku (required)
Path Parameter — SKU del producto a obtener cotización
plazo (required)
Path Parameter — plazo expresado en semanas

Return type

Cotizacion

Example data

Content-Type: application/json
{
  "abonoNormal" : 15790.45,
  "abonoPuntual" : 14900.32
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Cotización satisfactoria Cotizacion

404

No se encontró el plazo y/o el producto especificado
